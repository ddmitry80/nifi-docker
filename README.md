# nifi-docker

Простой инстанс NiFi и NiFi Registry, для учебных целей

Обычный запуск
```sh
docker compose up
```

## NiFi
url: http://localhost:18443/nifi/  

## Registry
docs: https://nifi.apache.org/docs/nifi-registry  
url: http://localhost:18080/nifi-registry  

### Подключение к registry
http://registry:18080
